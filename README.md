## Features 

**1.** Automatically adds arms to armor stands when placed

**2.** [1.21+] Shift + right click an armor stand to show / hide the arms

**3.** [1.21.6+] Settings with `/trigger armorstandarms.config`

  - Enable / disable the shift + right click feature

**Remove all arms**

```
/function armorstandarms:app/arms/remove_all
```

**Show all arms**

```
/function armorstandarms:app/arms/show_all
```

## Below 1.21 Features

If you want to **remove the arms** from **only one armor stand** run:

```
/trigger remove_arms
```
and to **enable** them again run

```
/trigger show_arms
```

For that to work without complications: Go as near as you can to the armor stand and use /trigger
